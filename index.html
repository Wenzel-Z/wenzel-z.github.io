Dining Philosophers

An implemented solution in Java of the dining philosophers problem as postulated by Edsger Dijkstra. The original problem is actually unsolveable, and requires some modifactions to prevent starvation or "deadlock". This solution utilizes a waiter or "mutex" to allow each philosopher to eat his/her fill. This article uses a different solution to solve the same problem
